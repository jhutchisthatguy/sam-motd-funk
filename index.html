Welcome to Funky gmod Server!

Server Rules

  • Listen to admins and mods
  • No server crashing or exploits
  • Respect fellow players
  • Have fun and enjoy the game!

Server Information

This is a sandbox casual server where you are free to do as you wish, within reason. Build, explore, and interact with others, but remember to respect the rules and the enjoyment of fellow players!

Server Commands

  • !motd – Shows the message of the day
  • !menu – Opens a limited SAM admin menu
  • !buddies – Opens the gProtect buddies menu so you can give your friends permission to interact with your props
